### 第二步: 部署到Netlify (15分钟)

#### 方法A: GitHub + Netlify (推荐)

```bash
# 1. 初始化Git仓库
cd /Users/smianmian/独立站/website
git init
git add .
git commit -m "Initial commit"

# 2. 推送到GitHub
# (在GitHub创建仓库后)
git remote add origin https://github.com/你的用户名/mewpow-website.git
git push -u origin main
```

然后:
1. 访问 https://www.netlify.com
2. 注册/登录
3. 点击 "Add new site" → "Import an existing project"
4. 选择 GitHub,授权
5. 选择你的仓库
6. 构建设置:
   - Build command: `npm run build`
   - Publish directory: `dist`
7. 点击 "Deploy site"

#### 方法B: 直接拖拽 (最简单)

1. 在本地运行: `npm run build`
2. 访问 https://app.netlify.com/drop
3. 把 `dist` 文件夹拖进去
4. 完成!

---

### 第三步: 绑定域名 (10分钟)

#### 在Netlify配置:
1. 进入你的网站设置
2. 点击 "Domain settings"
3. 点击 "Add custom domain"
4. 输入你的域名: `www.mewpow.com`

#### 在域名注册商配置DNS:

**添加以下DNS记录:**

```
类型    名称    值
A       @       75.2.60.5
CNAME   www     你的netlify域名.netlify.app
```

**等待生效** (5分钟-24小时)

#### 自动配置HTTPS:
- Netlify会自动配置免费SSL证书
- 等待5-10分钟即可启用HTTPS

#### 准备工作:
1. ✅ 购买域名
2. ✅ 域名备案 (15-20天)
3. ✅ 购买云服务器或对象存储

#### 方案A: 云服务器部署

```bash
# 1. 本地构建
npm run build

# 2. 连接服务器
ssh root@你的服务器IP

# 3. 安装Nginx
yum install nginx -y  # CentOS
apt install nginx -y  # Ubuntu

# 4. 上传文件
scp -r dist/* root@你的服务器IP:/usr/share/nginx/html/

# 5. 配置Nginx
vim /etc/nginx/nginx.conf
# (配置域名和SSL)

# 6. 启动Nginx
systemctl start nginx
systemctl enable nginx

# 7. 配置SSL (Let's Encrypt)
yum install certbot python3-certbot-nginx -y
certbot --nginx -d yourdomain.com
```

#### 方案B: 对象存储 + CDN

1. 创建对象存储桶
2. 上传 `dist` 文件夹内容
3. 配置静态网站托管
4. 绑定CDN
5. 配置域名
