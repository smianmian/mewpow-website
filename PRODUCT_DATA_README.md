# 产品数据生成脚本

此脚本用于从产品信息.md生成完整的多语言产品数据库

## 生成的产品特性:
- 完整的产品规格信息(型号、颜色、材质、功率等)
- 多语言支持(中文、英语、日语、德语、法语、西班牙语)
- 认证信息(CE/FCC/RoHS)
- 环保特性标注
- 创新技术亮点

## 产品分类:
1. 吹风机 (hairdryer)
2. 取暖器 (heater)
3. 烘干器 (shoedryer)
4. 毛球修剪器 (trimmer)
5. 加湿器 (humidifier)
6. 生活电器 (household)
7. 电动牙具 (toothbrush)
8. 空气净化器 (purifier)
9. 风扇 (fan)
10. 灭蚊灯拍 (mosquito)
