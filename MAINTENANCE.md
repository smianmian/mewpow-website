## 1️⃣ 更新产品信息

### 方法A：使用可视化工具（推荐）

1. **打开产品管理工具**
   - 在浏览器中打开：`http://localhost:5173/product-manager.html`
   - 或者在网站文件夹中双击 `product-manager.html`

2. **准备Excel文件**
   - 点击"下载Excel模板"按钮
   - 在Excel中填写产品信息：
     ```
     产品ID: humidifier-x1
     产品名称: X1 Ultrasonic Humidifier
     类别: humidifier
     图片路径: /products/humidifier/X1.jpg
     起订量: 1000
     规格1: 4L Capacity
     规格2: Ultrasonic
     规格3: Smart Control
     ```

3. **上传并转换**
   - 将Excel文件拖拽到上传区域
   - 检查预览数据是否正确
   - 点击"下载 products.json"

4. **替换文件**
   - 找到网站文件夹中的 `products.json`
   - 用新下载的文件替换它
   - 刷新网站查看效果

## 2️⃣ 更新联系方式

### 更新邮箱

1. 打开 `index.html` 文件
2. 搜索 `sales@mewpow.com`
3. 替换为新邮箱
4. 保存文件

### 更新WhatsApp

1. 打开 `index.html` 文件
2. 搜索 `wa.me/86138xxxxxxxx`
3. 替换为你的WhatsApp号码
   - 格式：国家代码 + 手机号（去掉开头的0）
   - 例如：中国手机 138 1234 5678 → `8613812345678`
4. 保存文件

### 更新地址

1. 打开 `index.html` 文件
2. 搜索 `Shenzhen, Guangdong, China`
3. 替换为实际地址
4. 保存文件

## 3️⃣ 更新公司介绍

### 更新中文介绍

1. 打开 `translations.json` 文件
2. 找到 `"zh"` 部分
3. 找到 `"about"` 下的 `"desc1"` 和 `"desc2"`
4. 修改文字内容
5. 保存文件

### 更新英文介绍

1. 在同一文件中找到 `"en"` 部分
2. 同样修改 `"about"` 下的内容
3. 保存文件

**提示**：西班牙语(es)和德语(de)的翻译建议使用专业翻译服务。

## 4️⃣ 添加新产品图片

### 步骤1：准备图片

- **格式要求**：JPG 或 PNG
- **背景要求**：纯白色背景
- **尺寸建议**：800x800 像素或更大
- **文件大小**：每张图片 < 200KB

### 步骤2：放置图片

1. 打开 `public/products/` 文件夹
2. 根据产品类别选择对应文件夹：
   - 加湿器 → `humidifier/`
   - 吹风机 → `hairdryer/`
   - 取暖器 → `heater/`
   - 等等...
3. 将图片复制到对应文件夹
4. 图片命名建议：`产品型号.jpg`（例如：`X1.jpg`）

### 步骤3：更新产品数据

使用产品管理工具（见第1步）添加新产品信息。

3. 点击 "Forms" 标签
4. 查看所有提交记录

## 6️⃣ 更新产品目录PDF

### 步骤1：准备PDF

- 文件名：`catalog-2025.pdf`
- 文件大小：建议 < 5MB
- 内容：产品图片、参数、价格表等

### 步骤2：上传PDF

1. 打开 `public/` 文件夹
2. 将PDF文件复制到这里
3. 确保文件名为 `catalog-2025.pdf`

### 步骤3：启用下载功能

1. 打开 `main.js` 文件
2. 搜索 `// window.open('/catalog-2025.pdf'`
3. 删除前面的 `//`，变成：
   ```javascript
   window.open('/catalog-2025.pdf', '_blank');
   ```
4. 保存文件

## 🆘 紧急联系方式

### 网站无法访问

1. **检查域名是否过期**
   - 登录域名注册商网站
   - 查看域名状态
   - 及时续费

2. **检查托管服务**
   - 登录 Netlify 或 Vercel
   - 查看部署状态
   - 查看错误日志

### 表单收不到邮件

1. **检查表单服务**
   - 登录 Formspree/Web3Forms
   - 确认服务正常
   - 检查邮箱设置

2. **检查垃圾邮件**
   - 查看垃圾邮件文件夹
   - 将发件人添加到白名单

### 图片不显示

1. **检查图片路径**
   - 确认图片在正确的文件夹
   - 检查文件名是否匹配

2. **检查图片格式**
   - 确保是 JPG 或 PNG 格式
   - 文件名不要包含中文或特殊字符

## 💡 实用技巧

### 快速测试网站

1. 在不同设备上打开网站：
   - 电脑（Chrome、Safari、Edge）
   - 手机（iOS、Android）
   - 平板

2. 测试所有语言版本：
   - 点击右上角语言切换按钮
   - 确认翻译正确

3. 测试表单：
   - 填写测试信息
   - 提交后检查是否收到邮件

### 图片优化工具

推荐使用以下免费工具压缩图片：
- **TinyPNG**: https://tinypng.com
- **Squoosh**: https://squoosh.app
- **Compressor.io**: https://compressor.io

### 翻译工具

- **DeepL**: https://www.deepl.com （质量最好）
- **Google Translate**: https://translate.google.com
- **专业翻译**：重要内容建议找专业翻译
